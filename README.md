## ğŸ› ï¸ Installation
### Prerequisites
- **Node.js**: `v18.0.0` or higher
- **npm**: `v8.0.0` or higher
- **Git**

### Setup
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/subh37106/thunder.git](https://github.com/subh37106/thunder.git)
    cd thunder
    ```

2.  **Install root dependencies:**
    ```bash
    npm install
    ```

3.  **Setup Frontend:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

4.  **Setup Backend:**
    ```bash
    cd be
    npm install
    npm run start
    ```
